# Core Personal Pronouns

| English        | Kuteta Root | IPA       |
| -------------- | ----------- | --------- |
| I / me         | ani         | [a.ni]    |
| you (singular) | tuya        | [tu.ja]   |
| he / she / it  | sena        | [se.na]   |
| we             | anilo       | [a.ni.lo] |
| you (plural)   | tulo        | [tu.lo]   |
| they           | seno        | [se.no]   |

---

# Pronouns in Use

## Subject Position

Use the **topic marker `li`** after the pronoun:

> **ani li kulo-ta** → “I spoke.”  
> **tuya li kulo-no** → “You will speak.”

## Object Position

Use the **object particle `na`** after the pronoun:

> **remi li tuya na miya-ta** → “The person saw you.”

# Demonstratives

Kuteta uses **standalone roots** for demonstratives, placed **before the noun** they modify.

## Demonstrative Roots

| English    | Kuteta Root | IPA     |
| ---------- | ----------- | ------- |
| this       | neta        | [ne.ta] |
| that       | sena        | [se.na] |
| that (far) | tona        | [to.na] |
| these      | neto        | [ne.to] |
| those      | seto        | [se.to] |

## Examples
- _neta nako_ → “this fire”
- _sena neko_ → “that cat”
- _tona selo_ → “that tree (far)”
- _neto nemi na miya-ta_ → “(I) saw these cats”
- _seto li tari-ta_ → “Those [things] fell”
---

# Reflexive Pronouns

Reflexives are formed by adding the **suffix `-ki`** to the base pronoun.

## Reflexive Forms

|Base Pronoun|Reflexive|Meaning|
|---|---|---|
|ani|ani-ki|myself|
|tuya|tuya-ki|yourself|
|sena|sena-ki|himself/herself|
|anilo|anilo-ki|ourselves|
|tulo|tulo-ki|yourselves|
|seno|seno-ki|themselves|

## Example Usage
- _ani-ki na miya-ta_ → “I saw myself”
- _tuya-ki na kulo-ta_ → “You spoke to yourself”
- _sena li sena-ki na miya-ta_ → “He saw himself”
